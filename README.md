# Photo Studio Management System - Frontend

A modern web application built with React and Vite for managing photo studio operations.

## Features

* User Authentication
* Customer Management
* Dashboard Analytics
* Responsive Design
* Secure API Integration
* Studio Management

## Technologies Used

* React.js
* Vite
* Redux Toolkit
* React Router
* Axios
* CSS

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and add:

```env
VITE_API_URL=YOUR_BACKEND_API_URL
```
